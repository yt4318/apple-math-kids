# 🍎 Apple Math Kids

子供向けのりんごを使った足し算ゲームです。ブラウザで遊べます。

## 遊び方

1. スタートボタンを押してゲーム開始
2. 1〜9の数字の足し算が出題されます
3. 数字の下にはりんごの絵が個数分ピラミッド型で表示されます
4. 4つの数字カードから正解を選んでタップ
5. 正解すると ○（ピンポン音）、不正解だと ×（ブー音）が表示されます
